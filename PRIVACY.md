# Privacy Policy

## Git Repositories for Command Palette

**Last updated:** March 24, 2026

### Overview

Git Repositories for Command Palette is a local-only extension that helps you discover and access git repositories on your computer. Your privacy is important to us.

### Data Collection

**This extension does not collect, store, transmit, or share any personal data.** Specifically:

- **No telemetry** — No usage data, analytics, or crash reports are collected
- **No network access** — The extension operates entirely offline and makes no network requests
- **No cloud storage** — All settings are stored locally on your device
- **No personal information** — No names, email addresses, or identifiers are gathered

### Local Data

The extension stores the following data locally on your device only:

- **Settings** (scan paths, max depth, terminal command) — saved to a local JSON file in your app data folder
- **Repository scan results** — held in memory only while the extension is running, never written to disk

### Permissions

The extension requires `runFullTrust` to scan your filesystem for git repositories and launch applications (Explorer, VS Code, Terminal). It does not use this capability to access data beyond what is needed to provide its features.

### Third-Party Services

This extension does not integrate with or send data to any third-party services.

### Changes

If this privacy policy changes, the updated version will be posted in this repository.
